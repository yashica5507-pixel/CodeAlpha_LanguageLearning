# **App Name**: LinguaLeap

## Core Features:

- Language Selection: Allow users to select the language they want to learn from a list of available languages.
- Daily Lessons: Provide daily lessons with new words, phrases, and sentences. Lessons should have a progress tracking mechanism
- Flashcards with Pronunciation: Display flashcards with translations and pronunciations for each new word or phrase.
- AI Pronunciation Tool: Provide the ability to record your attempt to pronounce a word and have AI score your accuracy.
- Quizzes and Practice Tests: Allow users to take quizzes and practice tests to check their learning progress and reinforce what they've learned.
- Progress Tracking: Visually display the user's progress through the lessons and quizzes, offering insights into their strengths and weaknesses.
- User Data Storage: Store user data and learning progress in Firestore database.

## Style Guidelines:

- Primary color: HSL 210, 75%, 60% - A vibrant blue (#39A7FF) to represent intelligence and clarity, creating a welcoming and engaging atmosphere.
- Background color: HSL 210, 20%, 95% - Very light blue (#F0F8FF) for a clean and unobtrusive background.
- Accent color: HSL 180, 60%, 40% - A contrasting teal (#26A69A) to highlight interactive elements and important information, promoting user engagement.
- Headline font: 'Poppins', sans-serif. Body font: 'PT Sans', sans-serif. Use Poppins for headlines, PT Sans for body.
- Use clean, modern icons to represent different categories (vocabulary, grammar, etc.)
- Maintain a clean and intuitive layout to ensure easy navigation and user-friendliness.
- Incorporate subtle animations and transitions to provide feedback and enhance the user experience.
